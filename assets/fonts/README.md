# Fonts

`NotoSansSC-VF.ttf` is included so the Web export can render Simplified Chinese text inside the Godot canvas. Godot Web does not reliably have access to the user's operating-system CJK fonts, so relying on `ThemeDB.fallback_font` can render Chinese as missing-glyph boxes.
